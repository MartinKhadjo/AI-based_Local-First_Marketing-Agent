# Demo Video

Public demo:

[AI-based Local-First Marketing Agent on YouTube](https://youtu.be/0TN4MeALWmg?si=rEkr6Z4LcSghJnsq)

## What the demo shows

The demo presents the Marketing Agent as a full campaign operating system:

1. Agent Chat command center.
2. Content-fundus Campaign Autopilot.
3. Platform credentials and website stores.
4. Product-growth analysis and landing-page strategy.
5. New campaign creation.
6. Post package approval.
7. Media asset library.
8. Processing pipeline.
9. Campaign preflight simulation.
10. Paperclip agent control plane.
11. Workspace governance and account context.

## Public-safe interpretation

The video shows the product workflow and architecture surface. It does not publish source code, secrets, OAuth tokens, private prompts, or customer data.

The demo should be understood as a portfolio walkthrough of a proprietary system:

- local-first campaign orchestration;
- governed AI content packaging;
- human-in-the-loop approval;
- optional MiroFish simulation;
- optional Paperclip control plane;
- external LLM/MCP orchestration;
- platform credential configuration;
- scheduler and publishing safety model.
