# Assets

This folder contains public-safe screenshot assets for the AI-based Local-First Marketing Agent portfolio repository.

The demo-video thumbnail is stored at the repository root as `maxresdefault.jpg`.

The product preview screenshot is stored at the repository root as `Marketing_Agent_Screenshot.png`.

The assets are used for demonstration and portfolio purposes only. They do not contain source code, platform secrets, OAuth tokens, private prompts, or customer data.
