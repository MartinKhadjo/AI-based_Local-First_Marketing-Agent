# Diagram Provenance

The diagrams in this repository are public-safe Mermaid diagrams derived from the professional system manuals and verified runtime documentation of the private AI-based Local-First Marketing Agent.

The diagrams intentionally describe:

- component boundaries;
- runtime responsibilities;
- operator workflows;
- MCP and external-agent handoff;
- simulation flow;
- scheduler flow;
- security and trust boundaries.

The diagrams intentionally do not expose:

- production source code;
- private prompt templates;
- credential values;
- internal adapter implementations;
- local machine secret paths;
- proprietary schemas;
- deployable service configuration.

Screenshots were selected from the public demo-video review frames and are included only as portfolio UI evidence.
